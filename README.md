# NODEJS TYPEORM COCKROACHDB TODOAPP EXAMPLE

Steps to run this project:

1. Run `npm i` command.
2. Setup database settings inside `ormconfig.json` file.
3. Run `npm start` command.
4. Run http://localhost:3000/ in your browser.
